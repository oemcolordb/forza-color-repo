🎨 FORZA COLOR UNIVERSE - PORTABLE VERSION

📁 QUICK START:
1. Double-click "portable-server.bat"
2. Your browser will open automatically
3. Enjoy all 10,000+ colors offline!

💻 REQUIREMENTS:
- Windows PC
- Any modern web browser
- No internet needed after setup

🌐 FEATURES:
- Browse 10,000+ official automotive colors
- Search by manufacturer, model, year
- Color matching from images
- Favorites system
- Dark/Light themes
- Mobile responsive

🔧 TROUBLESHOOTING:
If the batch file doesn't work:
1. Install Node.js from nodejs.org
2. Open Command Prompt in this folder
3. Run: npx http-server app -p 8080 -o

Enjoy exploring automotive colors! 🚗✨
