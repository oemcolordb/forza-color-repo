# Forza Color Universe - Mobile App

## How to use:

1. **For local testing:**
   - Install a local server (like Live Server extension in VS Code)
   - Open index.html with the server
   - Or use Python: `python -m http.server 8000`

2. **For sharing:**
   - Upload this entire folder to any web hosting service
   - Share the URL with your friend

3. **Free hosting options:**
   - Netlify (drag & drop the folder)
   - Vercel
   - GitHub Pages
   - Firebase Hosting

## Mobile-friendly features:
- Responsive design with Tailwind CSS
- Touch-friendly interface
- Works on all modern mobile browsers
