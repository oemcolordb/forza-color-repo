🎨 Forza Colors API

Public API for accessing 10,000+ automotive paint colors from Forza racing games.

Base URL

https://forza-colors.netlify.app/.netlify/functions/

Endpoints

GET /colors

Get automotive colors with optional filtering

Parameters:

  • make - Filter by manufacturer (e.g., Ferrari, BMW)
  • type - Filter by color type (e.g., Metal Flake, Normal)
  • search - Search color names
  • limit - Limit results (default: 100, max: 1000)

Example:

GET /colors?make=Ferrari&limit=10

GET /makes

Get all available car manufacturers

Example:

GET /makes

GET /stats

Get database statistics and top manufacturers

Example:

GET /stats

Response Format

{
  "count": 10,
  "colors": [
    {
      "make": "Ferrari",
      "model": "F40",
      "year": 1987,
      "colorName": "Rosso Corsa",
      "colorType": "Normal",
      "color1": { "h": 0, "s": 0.8, "b": 0.9 },
      "color2": { "h": 0, "s": 0.8, "b": 0.9 }
    }
  ]
}
