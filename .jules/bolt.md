## 2025-06-22 - List Lookups and React.memo Invalidations
**Learning:** We discovered a significant performance bottleneck in `SimpleColorGrid` and `VirtualColorGrid`. When rendering a large list of `ColorCard` components, passing inline functions (`() => onToggleFavorite(colorId)`) as props broke the `React.memo` wrapping on `ColorCard`, causing unnecessary O(N) re-renders across the grid whenever state updated. Additionally, using `favorites.includes(colorId)` resulted in an O(N) lookup inside the `.map` loop, turning it into an O(N²) operation overall for the list mapping.
**Action:** Always derive complex structures like lookup Sets from arrays using `React.useMemo` to ensure O(1) loop checks. Instead of passing inline function wrappers, move the contextual variable mapping (like `colorId`) inside the child component itself so you can pass the stable function reference from the parent, keeping `React.memo` optimizations intact.
