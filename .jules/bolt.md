## 2026-06-19 - Deferring expensive client-side filtering\n**Learning:** In a React application with a large dataset (e.g., thousands of colors in a single view), filtering lists synchronously on every keystroke can block the main thread and cause noticeable input lag.\n**Action:** Use `useDeferredValue` to defer the heavy filtering calculation, keeping the input highly responsive while performing the expensive operation in the background.
